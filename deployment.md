# Deployment Blueprint

## Local development

```bash
npm install
npm run dev
```

## Static deployment

```bash
npm run build
```

Deploy the `dist/` folder to GitHub Pages or an internal static host.

## On-premise production direction

For BEL deployment, package the platform with:

- Source code repository handover
- Dependency inventory
- Offline build instructions
- Synthetic-to-real data connector documentation
- Role-based access control design
- Audit logging design
- VAPT readiness checklist
- Secure solver adapter framework
- HPC/GPU queue adapter blueprint
